
# Личный проект «Кэт энерджи»

Cat Energy - это адаптивный веб-сайт интернет-магазина, специализирующегося на продаже функционального питания для котов. Проект включает в себя главную страницу с информацией о продуктах и программах питания, а также каталог товаров с возможностью заказа.

Основные функции
Адаптивный дизайн - корректное отображение на мобильных устройствах, планшетах и десктопах

Интерактивное меню - выпадающая навигация для мобильных устройств

Каталог продукции - с фильтрацией по типам питания (PRO для набора массы, SLIM для похудения)

Интерактивный слайдер - демонстрация результатов использования продукции

Интеграция с картами - отображение местоположения компании через Яндекс.Карты

Форма подписки - для получения информации об акциях и новинках

Технологии

HTML5 - семантическая разметка с использованием БЭМ методологии

CSS3 - адаптивная верстка с медиа-запросами

JavaScript (Vanilla) - интерактивность без сторонних библиотек

Особенности реализации
Адаптивная графика - использование <picture> с разными разрешениями и форматами (WebP/PNG)

SVG-спрайты - для иконок и векторной графики

CSS Grid и Flexbox - современные подходы к верстке

Mobile-first подход - разработка вначале для мобильных устройств

Прогрессивное улучшение - базовый функционал работает без JavaScript (навигация в mobile)

Оптимизация
Ленивая загрузка изображений (loading="lazy")

Веб-шрифты с правильным подключением и fallback'ами

Favicon разных размеров для различных устройств


Интеграции
Яндекс.Карты API - для отображения местоположения компании

Адаптивная карта - меняет размеры в зависимости от устройства
